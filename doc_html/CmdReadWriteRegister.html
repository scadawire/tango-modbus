Command ReadWriteRegister :
    Read and Write multiple 16bits registers.



ReadWriteRegister Definition
Input Argument Tango::DEVVAR_SHORTARRAY argin[0] = Read start address
argin[1] = Number of registers to read
argin[2] = Write start address
argin[3] = Number of registers to write
argin[4..n+3] = Register values
Output Argument Tango::DEVVAR_SHORTARRAY argout[0..n-1] = Register values
